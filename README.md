# Tin! Tilo! Rings!

チンチロリン（Cee-lo）をベースにした3リール回転式ゲーム。

## 遊び方

1. タイトル画面で9種類のルールから1つ選択
2. 3つのリングが回転し、順番に停止
3. 11本のラインで役を判定してスコア獲得
4. 目標スコア達成でクリア

## ルールタイプ

| カテゴリ       | ルール    | 内容                   |
| -------------- | --------- | ---------------------- |
| 時間制限       | rule*1*\* | 最速で目標スコア到達   |
| 回数制限       | rule*2*\* | 規定回数以内に目標達成 |
| スコアアタック | rule*3*\* | 制限時間内で最高スコア |

## 主な役

- **ピンゾロ**: [1,1,1] - 10倍
- **トリプルセブン**: [7,7,7] - 7倍 + 特殊効果
- **ゾロ目**: 同じ数字3つ - 2倍
- **シゴロ**: [4,5,6] - 2倍
- **カブ系**: mod 10で点数決定

## 特殊機能

- **バレットタイム**: リール速度低下
- **レボリューション**: スコア2倍
- **トリプルセブン効果**: ロールバック等

## セットアップ

```bash
git clone https://github.com/kako-jun/cee-lo-rings.git
cd cee-lo-rings
npm install
npm run dev
```

http://localhost:5173/cee-lo-rings/ で開く
