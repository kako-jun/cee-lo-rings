# Tin! Tilo! Rings!

チンチロリン（Cee-lo）をベースにした 3 リール回転式ゲーム。

## 遊び方

1. タイトル画面で 9 種類のルールから 1 つ選択
2. 3 つのリングが回転し、順番に停止
3. 11 本のラインで役を判定してスコア獲得
4. 目標スコア達成でクリア

## ルールタイプ

| カテゴリ       | ルール    | 内容                   |
| -------------- | --------- | ---------------------- |
| 時間制限       | rule*1*\* | 最速で目標スコア到達   |
| 回数制限       | rule*2*\* | 規定回数以内に目標達成 |
| スコアアタック | rule*3*\* | 制限時間内で最高スコア |

## 主な役

- **ピンゾロ** [1,1,1] — ×5
- **アラシカブ** [3,3,3] — ×3
- **ケモノ** [6,6,6] — ×-6
- **トリプルセブン** [7,7,7] — ×3 + 特殊効果
- **ゾロ目** (それ以外の同数3つ) — ×3
- **シゴロ** [4,5,6] — ×2
- **ヒフミ** [1,2,3] — ×-2
- **カブ系** mod 10 で点数決定

## 特殊機能

- **バレットタイム** — リール速度を 2 に固定
- **レボリューション** — multi 役の符号反転 (高スコア時に大ダメージ)
- **トリプルセブン効果** — 確率で all_1/all_6/triplets/all_123/all_456/rollback 等

## セットアップ

```bash
git clone https://github.com/kako-jun/cee-lo-rings.git
cd cee-lo-rings
npm install
npm run dev
```

http://localhost:3000/cee-lo-rings/ で開く

## 技術スタック

- **PixiJS v8**（レンダリング）
- **gsap**（Tween / アニメーション）
- **Howler.js**（オーディオ）
- **TypeScript + Vite**

## 実装の根拠

オリジナルの phina.js 版（`reference/tin-tilo-rings-phina/`）を正本として、PixiJS で再実装している。
判定ロジック・状態機械・スプライト座標は Phina 版と厳密一致。
